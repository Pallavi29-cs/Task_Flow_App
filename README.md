# Task Flow App

A simple Task Management Application built using React and Vite.

## Features

- User Login
- Add New Tasks
- Edit Tasks
- Delete Tasks
- Update Task Status
- Search Tasks
- Filter Tasks by Status
- Pagination Support
- Local Storage Data Persistence
- Responsive UI Design

## Technologies Used

- React JS
- Vite
- React Router DOM
- JavaScript
- CSS

## Setup Instructions

1. Clone the repository
git clone https://github.com/Pallavi29-cs/Task_Flow_App.git

2. Open the project folder
cd Task_Flow_App

3. Install dependencies
npm install

4. Start the development server
npm run dev

5. Open in browser
http://localhost:5173
